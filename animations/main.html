Animations

Animation Names:

  • fadeIn

    opacity changes from completely transparent to solid

  • scaleIn

    size changes from 0 to normal size

  • scaleDown

    height changes from 0 to normal size, while width stays the same

  • floatDown1/2

    position moves down from above, use floatDown2 for opacity change

  • floatUp1/2

    position up down from below, use floatUp2 for opacity change

  • floatLeft1/2

    position left down from right, use floatLeft2 for opacity change

  • floatRight1/2

    position right down from left, use floatRight2 for opacity change
